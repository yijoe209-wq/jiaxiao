# 📱 微信访问解决方案

## 问题说明

微信提示"如需浏览，请长按网址复制后使用浏览器访问"，这是因为：
1. Cloudflare Tunnel 的临时域名被微信标记
2. 微信对未备案域名有安全限制

---

## ✅ 解决方案

### 方法 1: 使用浏览器打开（当前最快）

**操作步骤：**

1. **长按链接**
   ```
   https://achievement-senior-any-manchester.trycloudflare.com/simulate
   ```

2. **选择"在浏览器中打开"**
   - 会跳转到系统浏览器（Safari/Chrome）

3. **继续测试**
   - 在浏览器中功能完全正常
   - 可以测试所有功能

---

### 方法 2: 使用本地网络测试（推荐）

如果您和测试手机在**同一个 Wi-Fi 网络**：

#### 步骤 1: 获取电脑的 IP 地址

```bash
# 在终端运行
ifconfig | grep "inet " | grep -v 127.0.0.1
```

会显示类似：`inet 192.168.1.100`

#### 步骤 2: 在微信中直接访问

假设您的 IP 是 `192.168.1.100`，在微信中访问：

```
http://192.168.1.100:5001/simulate
```

**优点：**
- ✅ 不被微信拦截
- ✅ 速度更快
- ✅ 更稳定的测试环境

**注意：**
- ⚠️ 只能在同一 Wi-Fi 下使用
- ⚠️ 使用 http（不是 https）

---

### 方法 3: 使用其他内网穿透服务

如果需要真正的微信内访问，可以尝试其他工具：

#### 选项 A: cpolar（国内推荐）

```bash
# 安装 cpolar
brew tap cpolar/homebrew-tap
brew install cpolar

# 启动（会生成不同的域名）
cpolar http 5001
```

生成的域名格式：`https://xxxx.cpolar.cn`

**在微信中的访问：**
```
https://xxxx.cpolar.cn/simulate
```

#### 选项 B: localtunnel

```bash
# 安装
npm install -g localtunnel

# 启动
lt --port 5001
```

生成的域名格式：`https://xxxx.localtunnel.me`

#### 选项 C: serveo（SSH 方式）

```bash
# 直接使用 SSH
ssh -R 80:localhost:5001 serveo.net
```

生成的域名格式：`https://xxxx.serveo.net`
