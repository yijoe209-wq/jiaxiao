# 作业助手 - 使用指南

## 🎯 产品定位

**核心价值：** 快速采集老师发布的作业信息，通过 AI 技术自动整理成结构化任务列表。

**使用场景：** 微信群接收老师作业 → 网页快速整理

---

## 📱 如何使用（推荐方案）

### 方案：收藏到微信

#### 第一次使用

1. **打开网页**
   ```
   https://achievement-senior-any-manchester.trycloudflare.com
   ```

2. **收藏到微信**
   - 点击右上角 "···" 菜单
   - 选择"收藏"
   - 完成！

#### 日常使用

**每次只需 3 步：**

1. **复制老师消息**
   - 在微信群长按老师消息
   - 点击"复制"

2. **打开收藏**
   - 微信 → 我 → 收藏
   - 找到"作业助手"
   - 点击打开

3. **粘贴提交**
   - 粘贴到输入框
   - 添加图片（如有）
   - 点击"AI 智能解析"
   - 选择学生并确认

---

## 💡 完整流程示例

### 场景 1：纯文字作业

```
老师在微信群发：
"语文作业：
1. 背诵《山行》古诗
2. 完成练习册第5页"

操作流程：
1. 长按消息 → 复制
2. 打开微信收藏 → 作业助手
3. 粘贴到输入框
4. 点击"AI 智能解析"
5. 查看结果 → 选择学生 → 确认
```

### 场景 2：图片 + 文字

```
老师在微信群发：
"数学作业，详见图片"
[图片1] [图片2]

操作流程：
1. 长按消息 → 复制文字
2. 保存图片到手机
3. 打开微信收藏 → 作业助手
4. 粘贴文字
5. 点击"选择图片" → 添加保存的图片
6. 点击"AI 智能解析"
7. 查看结果 → 选择学生 → 确认
```

## ❓ 常见问题

### Q: 为什么不能在微信内直接打开链接？

A: 微信安全限制，但可以通过"收藏"功能快速访问。

### Q: 如何收藏到微信？

A: 
1. 在微信中打开链接
2. 点击右上角 "···"
3. 选择"收藏"

### Q: 图片如何上传？

A: 
1. 长按图片 → 保存到手机
2. 在网页中点击"选择图片"
3. 选择保存的图片

### Q: AI 解析不准确怎么办？

A: 
1. 在确认页面手动修改
2. 提供反馈帮助优化
3. 后续会支持编辑功能

### Q: 可以管理多个学生吗？

A: 
- 当前版本支持 2 个学生
- 后续会支持动态添加
