## 📱 微信真实操作流程

### 在微信群中多选消息

#### 操作步骤:

1. **长按老师的消息**
   - 出现菜单选项

2. **选择"多选"**
   - 进入多选模式

3. **勾选多条消息**
   - 可以选择文本消息
   - 可以选择图片消息
   - 可以混合选择

4. **点击"转发"**
   - 选择转发目标
   - 发送到"文件传输助手"或其他

5. **复制内容**
   - 长按每条文字消息 → 复制
   - 长按每条图片 → 保存到手机

#### 在测试页面中使用:

1. **打开测试链接:**
   ```
   https://achievement-senior-any-manchester.trycloudflare.com/simulate
   ```

2. **粘贴所有文字**
   - 将复制的多条文字粘贴到输入框

3. **上传保存的图片**
   - 点击"选择图片"
   - 从相册选择保存的作业图片
   - 等待上传完成

4. **点击"模拟转发"**
   - 等待 AI 解析
   - 查看识别结果
   - 点击确认链接
