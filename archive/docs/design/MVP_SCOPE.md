# MVP 功能范围（调整版）

## 🎯 核心目标
**验证痛点：家长能否快速查看和确认孩子的任务**

---

## ✅ Phase 1: 核心功能（必须做）

### 1. 信息输入
- [x] 微信消息接收（文本）
- [ ] 微信消息接收（图片 + OCR）
- [ ] 批量转发支持

### 2. AI 解析
- [x] 单条任务解析
- [x] 复合任务解析（7条任务拆分）
- [x] 任务类型识别（阅读/背诵/书写等）
- [x] 置信度判断
- [ ] 降级策略（关键词匹配）

### 3. 用户确认
- [ ] 待确认任务列表展示
- [ ] 学生选择/分配
- [ ] 任务内容修正
- [ ] 批量确认

### 4. 任务查看
- [ ] Web 页面查看任务列表
- [ ] 按学生分类显示
- [ ] 任务详情展示
- [ ] 标记完成功能
- [ ] 任务筛选（全部/未完成/已完成）
