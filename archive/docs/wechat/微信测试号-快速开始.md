## ⚠️ 常见问题快速解决

### Q: 提交配置时提示"Token验证失败"

**A: 检查这3点：**
1. .env 中的 `WECHAT_TOKEN=jiaxiao123`
2. 测试号页面填写的 Token 也是 `jiaxiao123`
3. Flask 应用正在运行

```bash
# 确保应用运行
python app.py
```

---

### Q: 发送消息后没有回复

**A: 依次检查：**

1. **查看终端日志**
   - 是否有错误信息
   - 是否显示收到消息

2. **检查是否关注测试号**
   - 重新扫描二维码关注

3. **测试健康检查**
   ```bash
   curl http://localhost:5001/health
   ```

---

### Q: 点击链接打不开

**A: 长按链接选择"在浏览器中打开"**

或直接访问：
```
http://192.168.0.113:5001/tasks
```
